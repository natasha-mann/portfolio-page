PROJECTS

DishUp || Full-stack App

A meal planning app to make your life as simple as possible, whilst enjoying delicious food. DishUp offers users a way to easily search for food, using a range of possible filters for dietary requirements and intolerances. These meals can be saved to meal plans on the user profile, so they can be easily referred back to in the future.

GitHub Repo Live App

Travel Companion || Front-End Project

Travel Companion is a travel app designed to allow the user to search for a country of their choice, or view a randomly generated country, and find information such as places to see in the capital, health data and a currency converter. The user can add countries to their favourites through local storage to review at a later date.

GitHub Repo Live App

Conquering Code || MVC Project

This is a full stack project built with node.js, an express.js server, Sequelize database, handlebars views, HTML5, CSS3, Bootstrap, vanilla JavaScript and jQuery. It features user authentication, private routes and automatically expiring sessions. Authenticated users can write blog posts and comment on posts written by other users.

GitHub Repo Live App

Employee Tracker || Back-End Project

This app was built using Node.js and the inquirer package, with a MySQL database. Using the app, the user is able to build up data for a company, with tables for departments, job roles and employees. Once data has been added, the user can view the data tables in the terminal using the "view" options, or can choose to add to, update or delete the records.

GitHub Repo Demo Video

Team HTML Generator || Back-End Project

This project uses Node.js and the inquirer package to dynamically build a HTML document displaying employee detail cards. The app provides users with a series of prompts in the terminal, asking for information about their team, and uses JavaScript classes to generate various instances of the employee types. The app was developed using TDD and includes a comprehensive test package.

GitHub Repo Demo Video

Weather Dashboard || Front-End Project

This app uses the OpenWeatherAPI to retrieve weather forecast data. I used jQuery to dynamically render a current day overview and a 5-day forecast for any city searched. I used local storage to store the most recently searched places and rendered them on page dynamically. The page is styled using Bootstrap along with CSS3.

GitHub Repo Live App

Code Quiz || Front-End Project

This app used jQuery to dynamically render code questions on screen. The code is written so that the questions can easily be changed and/or added to. There are currently 3 sets of questions the user can choose to include in the quiz. This app uses local storage to store user scores and renders them on a separate html page in a highscores table.
