PROJECTS

Travel Companion

Travel Companion is a travel app designed to allow the user to search for a country of their choice, or view a randomly generated country, and find information such as places to see in the capital, health data and a currency converter. The user can add countries to their favourites through local storage to review at a later date.

GitHub Repo Live App

Weather Dashboard

This app uses the OpenWeatherAPI to retrieve weather forecast data. I used jQuery to dynamically render a current day overview and a 5-day forecast for any city searched. I used local storage to store the most recently searched places and rendered them on page dynamically.

GitHub Repo Live App

Code Quiz

This app used jQuery to dynamically render code questions on screen. The code is written so that the questions can easily be changed and/or added to. This app uses local storage to store user scores and renders them on a separate html page in a highscores table.

GitHub Repo Live App

README Generator

For this project I created a README generator, using Node.js and the inquirer package. The app provides users with a series of prompts in the terminal, asking for information about their project. With the user responses, the program generates a markdown file, with sections and a table of contents dynamically built based on which elements the user chooses to include.

GitHub Repo Demo Video

Team HTML Generator

This project uses Node.js and the inquirer package to dynamically build a HTML document displaying employee detail cards. The app provides users with a series of prompts in the terminal, asking for information about their team, and uses JavaScript classes to generate various instances of the employee types. The app was developed using TDD and includes a comprehensive test package.
